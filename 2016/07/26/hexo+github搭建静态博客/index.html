hexo+github搭建静态博客

hexo是一款基于Node.js的静态博客框架

hexo相关

安装

安装Node.js

Windows可以去Node.js官网下载安装包

CentOS可以yum安装

安装hexo

升级hexo

初始化BLOG

建立一个文件夹blog作为博客的文件夹

后面对于博客的操作都在此文件夹下进行

生成博客的文件

启动一个本地服务器，http://localhost:4000

编写blog

  1. 将写好的markdown文件放在blog/source/_posts文件夹下
  2. hexo g #生成文件
  3. hexo d #上传

github相关

首先需要在github上建立一个Repository，Repository名必须为your_user_name.github.io，固定写法your_user_name是你的用户名

然后找到你仓库的https地址

编辑blog目录下的_config.yml文件,修改或者新增一下内容

deploy:
type: git
repository: 你的https地址
branch: master

然后执行，可能时间有点长

然后执行

浏览器里输入your_user_name.github.io就可以看啦
